# Liri - a nodejs command line app  
## Purpose  
Liri is a node command line app that searches for songs, concerts, or movies based on queries issued to it through an Inquirer prompt system.

### Technology  
* Node JS
* Node Moduels
    * Axios
    * DotEnv
    * fs
    * Moment
    * Node-spotify-api
    * Inquirer
* API data
    * Seatgeek
    * OMDB
    * Spotify
